db Package¶

DB Module¶

This file contains a wrapper class to fetch data from a Sybase or MySQL database.

author:Sami-Matias Niemi (niemi@stsci.edu) for STScI.
class SamPy.db.DB.DBSMN(sql, user, password, database, address)[source]¶

Wrapper class to fetch data from a database. Separate functions for Sybase and MySQL database. A new class instance must be initialised for every server/database combination.

fetchMySQLData()[source]¶

Function to fetch data from MySQL database. Fetches all data from the database given in the class constructor. Returns the fetched data as an array.

fetchSybaseData()[source]¶

Function to fetch data from Sybase database. Fetches all data from the database given in the class constructor. Returns the fetched data as an array.
