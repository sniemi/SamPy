SamPy.astronomy.randomizers¶

randomizers Module¶

This module can be used to randomize for example galaxy positions.

depends:NumPy
author:Sami-Matias Niemi
date:21 May, 2011
SamPy.astronomy.randomizers.randomUnitSphere(points=1)[source]¶

This function returns random positions on a unit sphere. The number of random points returned can be controlled with the optional points keyword argument.

Param :the number of random points drawn
Ptype :int
Returns:random theta and phi angles
Return type:dictionary
