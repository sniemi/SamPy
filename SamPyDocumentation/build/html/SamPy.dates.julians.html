SamPy.dates.julians.toJulian(year, month, day, hour, minute, timezone='UTC')[source]¶

This function converts a given date to Modified Julian Date.

Note :The function uses Python’s inbuilt time module.
Param :year: year
Param :month: month
Param :day: days
Param :hour: hours
Param :minute: minutes
Param :timezone: Timezone, default is UTC
Returns:Modified Julian Date
Return type:float
