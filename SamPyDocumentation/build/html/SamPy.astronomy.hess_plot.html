SamPy.astronomy.hess_plot¶

hess_plot Module¶

This file contains a function that can be used to generate a Hess plot.

SamPy.astronomy.hess_plot.hess_plot(xdata, ydata, weight, xmin, xmax, nxbins, ymin, ymax, nybins, pmax=1.0, pmin=0.1)[source]¶

This function can be used to calculate a hess plot i.e. the conditional probability 2D histogram.

Parameters:
  • xdata – 1D NumPy array containing data in x dim.
  • ydata – 1D NumPy array containing data in y dim.
  • weight – 1D NumPy array containing weight for each data point
  • xmin – a minimum x value to be considered
  • xmax – a maximum x value to be considered
  • nxbins – the number of bins in x dim.
  • nybins – the number of bins in y dim.
  • pmax – 1.0
  • pmin – 0.1
Returns:

2D array, min, max
