SamPy.astronomy.differentialfunctions¶

differentialfunctions Module¶

Some functions to calculate differential functions. These can be used to calculate luminosity functions, stellar and dark matter halo functions etc.

requires:NumPy
version:0.15
author:Sami Niemi
contact:niemi@stsci.edu
SamPy.astronomy.differentialfunctions.diffFunctionLogBinning(data, column=0, log=False, wgth=None, mmax=15.5, mmin=9.0, nbins=35, h=0.7, volume=250, nvols=1, physical_units=False, verbose=False)[source]¶

Calculates a differential function from data. Uses NumPy to calculate a histogram and then divides. each bin value with the length of the bin. The log binning refers to 10 based log, i.e. log10(data)

SamPy.astronomy.differentialfunctions.diff_function_log_binning(data, column=0, log=False, wgth=None, mmax=15.5, mmin=9.0, nbins=35, h=0.7, volume=250, nvols=1, physical_units=False, verbose=False)[source]¶

Calculates a differential function from data. :warning: One should not use this, unless the number of systems for each bin is used. One should use diffFunctionLogBinning instead, which is probably faster as it uses NumPy.histogram rathrer than my own algorithm.
