SamPy.astronomy.basics¶

basics Module¶

This module contains basic information of different subjects often needed in astronomy such as conversions from one magnitude system to another.

author:Sami-Matias Niemi
contact:niemi@stsci.edu
version:0.01
SamPy.astronomy.basics.JohnsonToABmagnitudes()[source]¶

Returned dictionary values can be used to convert Johnson U, B, V, and K bands to Vega system.

Return type:dictionary
