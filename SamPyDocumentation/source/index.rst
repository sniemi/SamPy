Welcome to SamPy's documentation!
=================================

Introduction
============

:Author: Sami-Matias Niemi
:Contact: sniemi@unc.edu
:version: 0.1

SamPy is a collection of Python functions, classes, methods, and scripts which are mostly
related to Astronomy or Cosmology. It provides several subpackages each representing a
different theme or functionality. SamPy comes without any warranty that any of the
subpackages or their methods are correct. **Beware of Bugs!**


Installation
============

SamPy is not available for general public yet.
